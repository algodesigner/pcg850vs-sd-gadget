Sample Files
============

BASEN.BAS   This is the BASE-N program that executes when the BASE-N
            key is pressed.

picpro.asm  PIC program that runs on the prototype PCB and flashes the LED.

SCOPEDEMO.DAT A BASIC program that writes 0 to 255 to the PIO port. This
	      can be used to exercise the PIO scope option of the SD card
	      gadget. Set PIO mode and Scope PIO and then run this program.

ROMRD3.DAT    A BASIC program that attempts to read a 93C56 EEPROM.
	      Unfortunately it doesn't work.
